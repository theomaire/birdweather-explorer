Netherlands

What is this?

An interactive dashboard exploring bird acoustic detection data across the Netherlands. Browse activity rhythms, seasonal patterns, weather correlations, migration maps, behavioral clustering, and more for 116 bird species.

Data source

All detections come from the BirdWeather network — a global community of citizen-science stations running BirdNET-Pi devices that record and classify bird sounds 24/7.

Coverage

Time period: March 2025 – March 2026 (one full year).

Spatial extent: The Netherlands only (bounding box ~50.7–53.6°N, 3.3–7.2°E).

Species: 116 species with sufficient detection counts. Rare vagrants and species with very few detections are excluded.

How to use

Species Explorer — Select a species from the left panel and navigate the tabs: activity rhythm (relative to sunrise), daily counts, hour×date heatmaps, ML-derived insights, weather correlations, station maps, migration animation, and false-positive co-occurrence analysis.

Overview — Cross-species views: behavioral clustering (UMAP/PCA of activity profiles) and species prediction (available in the full version only).

Disclaimer

Detections are produced by BirdNET, a deep learning model for bird sound identification. While highly capable, acoustic ML classification is not perfect — expect some false positives and missed detections, particularly for rare species, nocturnal calls, or noisy environments. The "FP Analysis" tab explores co-occurrence patterns that may indicate systematic misidentifications. This data should not be treated as verified ornithological records.
